About

Siapa gue?

Gue Anugrah Louis, seorang fullstack developer yang nge-code dari HP — literally. Pakai Acode sebagai editor dan Termux sebagai Linux environment, gue build dan deploy website langsung dari mobile.

Fokus utama gue di HTML, CSS, JavaScript untuk frontend, dengan Firebase untuk backend (auth, realtime database), dan GitHub Pages untuk hosting zero-cost.

Sekarang lagi banyak ngoprek soal AI Agent & Automation — gimana caranya teknologi bisa kerja otomatis buat ngebantu orang banyak.
